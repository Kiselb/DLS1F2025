# Тема проекта: Face Recognition (Разработка пайплайна для задачи Face Recognition)

## Этапы проекта

### Обязательная часть:
1. Изучить литературу, разобраться в том, как работают системы распознавания лиц

   **Обзор литературы Face Recognition.ipynb**
   
2. Скачать сырой датасет с фотографиями людей, сделать предобработку
3. Реализовать нейронную сеть для детекции ключевых точек лица
4. При помощи полученной модели выровнять лица в датасете
5. Реализовать ArcFace и с помощью него обучить модель для распознавания лиц, сравнить с CE loss
6. Построить полный пайплайн для распознавания лиц, оформить репозиторий и выводы

### Дополнительные задания:
1. Реализовать метрику качества ID rate
2. Поставить эксперименты с другой лосс-функцией, например Triplet Loss
3. Обучить сеть на смеси лоссов
4. Провести ресерч по open-source готовым решениям для задачи Face Recognition

   **OSS ресурсы.ipynb**
